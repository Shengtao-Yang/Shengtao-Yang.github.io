我是杨圣涛，目前在[南京信息工程大学](https://www.nuist.edu.cn/)[自动化学院](https://auto.nuist.edu.cn/main.htm)攻读电气工程及其自动化学士学位。获2025年全国大学生电子设计竞赛A题[全国二等奖](https://www.nuedc-training.com.cn/index/news/details/new_id/340)，其他学科类竞赛8项。参与无线电能传输领域的研究（导师：[连静](https://faculty.nuist.edu.cn/lianjing/zh_CN/index.htm)），投稿Electronics(JCRQ2, IF=2.6)一作论文一篇（Under Review）。公开电能质量领域发明专利一件（第三作者）。主持大学生创新创业训练项目两项。担任自动化学院科学技术协会主席。获校三好学生、一等奖学金、优秀学生干部、年度先进个人等荣誉。

自大二以来参与电力电子方向的学习与实践。设计并制作包括Boost,Buck,Buck-Boost,单相/三相全桥逆变/整流，PFC等在内的电源模块，基于DSP部署相关控制算法（包括电流电压双闭环、下垂、主从、SOGI锁相环等）。能够使用嘉立创EDA、Altium Designer、MATLAB/Simulink、CCS、STM32CubeIDE、Ansys Maxwell等工程软件。在[知乎](https://www.zhihu.com/question/494947440/answer/2028242293434401520)发布电赛电源题经验帖获19赞同、81收藏。

A data-driven personal website
======
Like many other Jekyll-based GitHub Pages templates, Academic Pages makes you separate the website's content from its form. The content & metadata of your website are in structured Markdown files, while various other files constitute the theme, specifying how to transform that content & metadata into HTML pages. You keep these various Markdown (.md), YAML (.yml), HTML, and CSS files in a public GitHub repository. Each time you commit and push an update to the repository, the [GitHub pages](https://pages.github.com/) service creates static HTML pages based on these files, which are hosted on GitHub's servers free of charge.
